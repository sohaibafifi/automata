DotNetZip Reduced Library, Developers' Kit package, v1.7 packed Wed-03-04-2009--00924.41.   This is the reduced version of the DotNetZip library.  It includes the classes in the Ionic.Zip namespace as well as the classes in the Ionic.Zlib namespace.  The reduced library differs from the full library in that it lacks the ability to Save Self-Extracting archives, and is much smaller than the full library.
